--Italics--
we can use either *(astrics) or _ (underscore)

*This text* is italics
_This text_ is italics


--strong--
can use either **(double astrisk) or __(double underscore)

**This text** is strong
__This text__ is strong


--strikethrough--
we can use ~~(double tilde)

~~This text~~ is strikethrough


--horizontal rule--
we can use ---(tiple hiphen) or ___(triple underscore)
---
___


--use \ for escaping in case of * or others--




--blockquote--
we can use > (greater than sign)
> This is a quote


--Links--
for links text goes in square brackets[] and link goes in paranthesis()

[Rohit Kumar Portfolio](http://www.rohitkumar.com/resume.html)


--unorderred lists--
we can use *(single astrisk symbol)
if we want nested list we can just tab and start over

* Item 1
* Item 2
* Item 3 
	* Subitem 1
	* Subitem 2



--ordered list--
we can use 1 followed by .
1. Item 1
1. Item 2
1. Item 3
1. Item 4


--inline code block--
we can use `` (back ticks)

`<p>This is a paragraph</p>`


--images--
same as links, preceded by !
